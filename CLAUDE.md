# CLAUDE.md — Project Instructions

## Project: Biblical Storytelling Channel
**Directory:** 

**Purpose:** YouTube (8-12 min) + TikTok (60-90 sec) videos on little-known Bible stories

## Workflow Pipeline
1. Script (YouTube + TikTok versions)
2. Scene breakdown (JSON)
3. Image prompts (ComfyUI/SDXL)
4. Voiceover notes (ElevenLabs)
5. Assembly (CapCut / DaVinci Resolve)

## Hardware
- GPU: RTX 4060 (8GB VRAM)
- OS: Windows 11
- Local image gen via ComfyUI

## Content Rules
- Always cite scripture references (KJV primary, ESV secondary)
- Storytelling-first, not lecture style
- Biblically accurate — no speculation presented as fact
- Label any extra-biblical context clearly as "historical context" or "tradition holds"
- Hook within first 3 seconds for TikTok, first 30 seconds for YouTube
